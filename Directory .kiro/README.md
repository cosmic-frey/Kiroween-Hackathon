## Projects

### Tamaghosty
A self-care companion PWA themed around a resurrected Tamagotchi ghost.

**Location:** `/tamaghosty`

**Tech Stack:**
- React + Vite
- Progressive Web App (PWA)
- LocalStorage for data persistence

**Features:**
- Study timer with Pomodoro technique
- Hydration tracking
- Meditation & breathing exercises
- Reflective journaling
- Meal logging
- Interactive ghost pet companion

**Deployed:** Netlify/Vercel
